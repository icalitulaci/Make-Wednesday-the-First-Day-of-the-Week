# Make-Wednesday-the-First-Day-of-the-Week
A calender Generator based on HTML that can make any day of the week as the first day of the week, including wednesday.

Try it
https://icalitulaci.github.io/Make-Wednesday-the-First-Day-of-the-Week/

## Features
- Generate calender that starts from any day of the week.
- Integrate Public Holiday based on nager.date
- Set your own "weekends" days, as many as you like
- Generate almost at any year
- Custom colour scheme
- Export into PDF, Image, or CSV
